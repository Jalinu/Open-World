What is CSS?

CSS stands for Cascading Style Sheets. It is used to design the layout, colors, fonts, and overall visual appearance of your HTML code.

CSS Code Example: 

.my-style {
    color: #ff007f;
    font-size: 24px;
    font-weight: bold;
    border: 2px solid #ff007f;
    border-radius: 50px;
    background-color: #fff0f5;
}
