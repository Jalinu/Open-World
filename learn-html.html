What is HTML?

HTML stands for HyperText Markup Language. It tells the browser how to display content like texts, headings, links, and images.

Try This Code: 

<h1>Hello World!</h1>
<p>This is my very first webpage built with World Web.</p>
<button class="btn btn-warning">Click Me!</button>
